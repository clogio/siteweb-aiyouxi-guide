# siteweb-aiyouxi-guide

## 项目简介

本仓库用于归档和发布多个独立的 HTML 页面。这些页面以静态文件形式保存，不针对任何特定域名或网站，旨在提供一种轻量、可访问的页面存档方式。

## 目录说明

```
siteweb-aiyouxi-guide/
├── pages/          # 存放各个 HTML 页面文件
├── assets/         # 页面引用的资源文件（如样式、图片、脚本等）
└── README.md       # 本文件
```

- `pages/` 目录下每个 HTML 文件对应一个独立页面，文件名通常反映页面内容或用途。
- `assets/` 目录用于存放页面所需的公共资源，便于管理和复用。

## 页面归档说明

- 所有页面均为纯 HTML 格式，可直接在浏览器中打开查看。
- 页面内容保持相对稳定，不承诺实时更新或维护。
- 归档的页面可能涉及不同主题或功能，具体请参见文件名及页面内说明。

## 维护说明

- 本仓库由维护者不定期更新，添加新的页面或调整现有内容。
- 欢迎通过提交 Issue 或 Pull Request 提出建议或报告问题。
- 请注意，本仓库仅作归档用途，不提供任何形式的服务或支持。

## 使用方式

1. 克隆或下载本仓库到本地。
2. 使用浏览器打开 `pages/` 目录下的 HTML 文件即可查看对应页面。
3. 如需引用资源，请确保 `assets/` 目录中的文件路径正确。
